# Utilities (Helpers, Shared Tools, Miscellaneous)

This directory contains miscellaneous helper scripts and utilities used across
the Marithmetics project.

Typical contents include:
- shared mathematical helpers  
- numeric utilities  
- Fejér / Ω kernel helpers  
- plotting or formatting helpers  
- CLI utilities  
- debugging or profiling tools  

Everything here is optional but supports the main theoretical directories.
