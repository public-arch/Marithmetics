# Responsible Use Policy

Marithmetics is published to support reproducible scientific auditing and research.

## Policy statement
We ask users and downstream redistributors to refrain from using this project in ways that enable violence or physical harm to life.

## Not intended for
This project is not intended to be used to:
- design, develop, produce, or operate weapons or weapon systems;
- enable, facilitate, or optimize violence, lethal force, or physical harm to life;
- support military or intelligence targeting, kill-chain optimization, or tactical operations;
- build systems intended to intimidate, coerce, threaten, or unlawfully surveil individuals or groups in ways that can lead to harm.

## Reporting
If you believe this project is being used in a way that could cause harm, please open an issue in the repository describing the concern. If you prefer not to post publicly, contact the maintainers via the email listed in the repository profile.
